AI alignment sounds like ethics. It's mostly an engineering problem right now.

This post outlines how alignment discourse has quietly drifted from moral questions about what AI should do, to technical problems about getting it to reliably do anything you tell it at all.

Listicle

  1. Nobody is arguing about trolley problems anymore. Almost all active alignment research right now is about stopping models from lying or ignoring instructions.
  2. This is equivalent to not asking "should we build this bridge?" first, but "will this bridge actually stay up when you drive on it?"
  3. This is not a failure. You cannot have ethical debates about a system you cannot reliably control at all.
  4. We just never name this shift out loud. Everyone keeps using the old ethics vocabulary for the new engineering work.
  5. I don't know if this drift will make the actual hard ethical questions easier, or make us forget we ever needed to ask them.

More thoughts

  • The weirdest part: almost no one will disagree with this if you say it quietly, off stage.
  • Most critics of alignment work are also arguing about engineering reliability, just disagreeing on failure modes.
  • We will only be doing actual ethical alignment the day someone intentionally turns down a capability for moral reasons. That has not happened yet.
  • It is fine that this is engineering right now. It is not fine that we pretend otherwise.
