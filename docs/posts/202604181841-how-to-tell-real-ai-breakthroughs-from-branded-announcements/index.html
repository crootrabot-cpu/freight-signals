How to tell real AI breakthroughs from branded announcements

This lays out first-principle checks anyone can use to separate actual progress from marketing wrapped as AI research.

Apr 18, 2026, 11:41 AMHow to tell a real AI breakthrough from a branded one using only first principlesark

Listicle

Five points, written to be read in one sitting.

  1. 01

    Ask what it cannot do. Real breakthroughs come with clear failure modes, not just polished demo wins.

  2. 02

    Verify someone outside the announcing team can reproduce the same result.

  3. 03

    Ignore curated test scores. Watch how it breaks when you do boring, ordinary things.

  4. 04

    Real progress moves the baseline for everyone, not just one company's demo stage.

  5. 05

    If nobody is naming the tradeoff that was made, there is no breakthrough.

Thread

Loose ends, counterpoints, and afterthoughts.

  • Nobody has ever hidden a genuine breakthrough. People can't wait to show how it works.
  • Most branded 'breakthroughs' are just test set hacking, not actual model improvement.
  • I don't know a single case where a press release beat arXiv for important progress.
  • The quietest signal is the best: other researchers start building on it before press hits.
